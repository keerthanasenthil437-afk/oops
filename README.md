# Quiz Application using Java Swing

Author: Keerthana S

## Aim
To develop a Java program using Swing components that conducts a multiple-choice quiz with a countdown timer for each question.

## Description
This project is a simple quiz application created using Java Swing.  
The program displays multiple-choice questions and provides four options for each question.  
A timer of 30 seconds is given for every question.

## Features
- Multiple choice questions
- 30-second countdown timer
- Next button to move to next question
- Automatically moves to next question when time ends
- Displays "QUIZ FINISHED!!!" after all questions

## Technologies Used
- Java
- Swing GUI
